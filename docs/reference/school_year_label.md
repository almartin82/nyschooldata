# School Year Label

Converts an end year integer to a school year label string.

## Usage

``` r
school_year_label(end_year)
```

## Arguments

- end_year:

  Integer end year (e.g., 2024)

## Value

Character school year label (e.g., "2023-24")

## Examples

``` r
school_year_label(2024)  # Returns "2023-24"
#> [1] "2023-24"
school_year_label(2021)  # Returns "2020-21"
#> [1] "2020-21"
```
