# Download raw school directory data from SEDREF COGNOS

Downloads the CEO contact information report from NYSED's SEDREF COGNOS
public reports portal. This report includes schools, districts, charter
schools, and nonpublic schools.

## Usage

``` r
get_raw_directory(end_year)
```

## Arguments

- end_year:

  School year end (not used - SEDREF provides current data)

## Value

Data frame with raw directory data from SEDREF
